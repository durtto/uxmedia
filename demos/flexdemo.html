Ext.ux.Media.Flex

Welcome to version 1.0 of the Ext.ux.Media.Flex classes.

ux.Media.Flex implements the FABridge functionality to interact directly with embedded Flex applications.

Classes provided (xtype(s))

  • ux.Media.Flex (the base Flex class from which all others are derived)
  • ux.Media.Flex.Component (flex, uxflex)
  • ux.Media.Flex.Panel (flexpanel)
  • ux.Media.Flex.Portlet (flexportlet)
  • ux.Media.Flex.Window
